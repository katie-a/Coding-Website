The Pet Pigeon Game


Description
  • Language: Python
  • Modules: Pygame, Time, Random
  • Link to code on GitHub

This is a game written in Python wherein you must care for a pigeon. This was my little sister's idea, inspired her great love of pigeons. I am currently making another version that has the same gameplay but the code is more organised. The user experience won't change but I, the developer, will be happier. Most, if not all, of the artwork has been painstakingly drawn by me in Paint. Without a mouse.
