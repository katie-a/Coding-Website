Hangman


Description
  • Language: Python
  • Modules: Pygame, Random
  • Link to code on GitHub

This is a hangman game written in Python with the help of Pygame and Random. I started off by making a command line version and gradually transforming it into a 2D game. If you want to run this you'll need a working installation of Pygame and Python. Otherwise, if you only have Python, you can run the command line version. P.S. My sister helped me come up with the wordlist, so expect supermoons, pigeons, and bassethounds. And pythons, of course.
