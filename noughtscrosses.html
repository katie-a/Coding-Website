Noughts And Crosses


Description
  • Language: Python
  • Modules: Pygame
  • Link to code on GitHub

This noughts and crosses game is probably the first game I ever made with GUI. I'm especially proud of the GUI, I find it very aesthetically pleasing. Just now it is human vs human (or cat versus human, it's really up to you), but one day I hope to make a human vs computer version. You will need pygame to run this. And, preferably, someone (or something) else to play against, unless you want to draw every time.
